# Данные

Каталог разделен на три части:

- `raw/` — исходные данные;
- `sample/` — небольшие учебные подвыборки и демонстрационные файлы;
- `processed/` — результаты предобработки.

## Учебные тематические наборы

В `data/sample/` размещены локальные файлы, на которых должны запускаться блокноты после обычного клонирования репозитория:

- `classification/electrical_grid_stability.csv`
- `regression/combined_cycle_power_plant.csv`
- `time_series/household_power_hourly_january_2007.csv`

Эти файлы подобраны так, чтобы:

- открываться без ручной настройки путей;
- использоваться через относительные пути внутри блокнотов;
- обеспечивать воспроизводимый запуск основных учебных примеров.

## Правила

- не хранить в git крупные сырые выгрузки без необходимости;
- для публичных источников фиксировать URL и способ получения в `docs/datasets.md`;
- для учебных ноутбуков по возможности использовать `sample/` или автоматически воспроизводимую загрузку.

## Восстановление полных данных

Для повторной загрузки открытых наборов данных использовать:

```bash
python scripts/fetch_open_datasets.py
```

Скрипт сохраняет полные версии во `data/raw/external/` и подготавливает учебные подвыборки в `data/sample/`.
