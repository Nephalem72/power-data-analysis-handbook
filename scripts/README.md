# Scripts

Каталог содержит служебные сценарии, необходимые для воспроизводимой работы с открытыми наборами данных.

## Доступные сценарии

- `fetch_open_datasets.py` — загрузка открытых наборов данных из UCI, сохранение полных версий в `data/raw/external/`, подготовка локальных учебных выборок в `data/sample/` и формирование демонстрационных файлов разных форматов в `data/sample/formats/`.

## Запуск

```bash
python scripts/fetch_open_datasets.py
```
