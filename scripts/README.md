# Scripts

Каталог содержит служебные сценарии, необходимые для воспроизводимой работы с открытыми наборами данных.

## Доступные сценарии

- `fetch_open_datasets.py` — загрузка открытых наборов данных из UCI, сохранение полных версий в `data/raw/external/` и подготовка локальных учебных выборок в `data/sample/`.

## Запуск

```bash
python scripts/fetch_open_datasets.py
```
