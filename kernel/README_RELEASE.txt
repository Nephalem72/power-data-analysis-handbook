Power Data Analysis Kernel

Contents:
- install_power_data_kernel.bat
- launch_jupyter_lab.bat
- power_data_kernel_requirements.txt

Usage:
1. Run install_power_data_kernel.bat
2. Wait until the environment is created and the kernel is registered
3. Start JupyterLab with launch_jupyter_lab.bat or from your own Jupyter installation
4. Select the kernel "Power Data Analysis"
