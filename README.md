# Системный анализ данных для электроэнергетиков в Jupyter-блокнотах

Репозиторий учебно-методического комплекта по прикладному анализу данных в электроэнергетике.
Здесь собираются пособие, Jupyter-блокноты, демонстрационные данные, задания и сопроводительная документация.

Публичный репозиторий:
[https://github.com/Nephalem72/power-data-analysis-handbook](https://github.com/Nephalem72/power-data-analysis-handbook)

## Для кого этот проект

Материалы предназначены для:

- студентов и магистрантов направлений, связанных с электроэнергетикой;
- инженеров, которым нужен воспроизводимый рабочий контур анализа данных;
- преподавателей, которым нужен связанный комплект "пособие + ноутбуки + задания + данные".

## Что входит в комплект

- учебное пособие по системному анализу данных;
- ноутбуки по разделам курса;
- три сквозных кейса:
  классификация, регрессия, временные ряды;
- демонстрационные датасеты и инструкции их получения;
- задания для самостоятельной работы;
- документация по запуску, структуре и воспроизводимости.

## Логика курса

Курс строится не как обзор Python или Jupyter самих по себе, а как прикладной маршрут:

1. рабочая среда и базовые принципы работы в Jupyter;
2. минимум Python для анализа данных;
3. табличные данные и форматы источников;
4. предобработка и разведочный анализ;
5. подготовка признаков и метрик;
6. введение в машинное обучение;
7. базовое знакомство с нейросетями на табличных данных;
8. воспроизводимость исследования.

## Быстрый старт

1. Установить Python 3.11+.
2. Создать виртуальное окружение.
3. Установить зависимости:

```bash
pip install -r requirements.txt
```

4. Запустить JupyterLab или Jupyter Notebook:

```bash
jupyter lab
```

или

```bash
jupyter notebook
```
