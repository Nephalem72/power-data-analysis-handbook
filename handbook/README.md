# Пособие в LaTeX

В каталоге `handbook/latex/` находятся:

- `main.tex` — основной файл пособия;
- `preamble.tex` — преамбула и служебные команды;
- `chapters/` — текст глав;
- `FIGURE_PLAN.md` — план рисунков и скриншотов;
- `main.pdf` — текущая собранная версия пособия.

## Сборка

Рекомендуемая команда:

```bash
latexmk -xelatex main.tex
```

Рабочая директория для сборки:

```text
handbook/latex
```

Используется XeLaTeX, поскольку пособие содержит русскоязычный текст.
