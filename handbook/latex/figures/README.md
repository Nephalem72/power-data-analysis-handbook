# Figures

Этот каталог предназначен для:

- скриншотов JupyterLab и блокнотов;
- рисунков и схем, которые будут вставляться в LaTeX-пособие;
- экспортированных графиков из блокнотов.

Пока рисунки не добавлены, в LaTeX-главах используются явные placeholder-вставки с инструкциями:
что именно необходимо снять, какую подпись использовать и какой текст нужно написать после рисунка.
