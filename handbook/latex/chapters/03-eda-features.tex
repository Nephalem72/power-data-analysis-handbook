\chapter{Разведочный анализ и формирование признаков}

\section{Роль разведочного анализа}

Разведочный анализ данных необходим для того, чтобы перейти от формального наличия таблицы к содержательному пониманию исследуемого процесса.
В задачах электроэнергетики это особенно важно, поскольку за числовыми рядами обычно стоят реальные режимные процессы, ограничения оборудования и временные закономерности.

Разведочный анализ позволяет:
\begin{itemize}[leftmargin=1.25cm]
\item оценить диапазоны и разброс значений;
\item выявить выбросы и аномальные режимы;
\item установить связи между признаками;
\item проверить наличие суточной, недельной или сезонной структуры;
\item выбрать признаки, которые будут использоваться на следующем этапе.
\end{itemize}

\section{Описательные статистики}

Первый шаг состоит в расчете описательных статистик.
Для числовых признаков обычно анализируются среднее значение, медиана, стандартное отклонение, минимум и максимум.
Для категориальных признаков интерес представляют число уникальных значений и частоты категорий.

Однако сами по себе такие показатели не дают полного ответа.
Они должны сопоставляться с физическим смыслом данных.
Например, допустимый диапазон напряжения, тока или мощности не является только статистической характеристикой --- он связан с режимными и нормативными ограничениями.

\section{Признаковое пространство}

Для дальнейшего моделирования недостаточно ``сырых'' наблюдений.
Необходимо построить признаковое пространство, которое отражает структуру процесса и может быть подано на вход модели.

На практике это означает формирование:
\begin{itemize}[leftmargin=1.25cm]
\item календарных признаков;
\item лаговых признаков;
\item агрегатов по окнам наблюдения;
\item относительных и нормированных показателей;
\item инженерно содержательных производных характеристик.
\end{itemize}

\section{Лаговые и агрегированные признаки}

Особое значение в задачах прогнозирования имеют лаговые признаки.
Если обозначить нагрузку как $L_t$, то лаговый признак первого порядка может быть записан как
\[
L_{t-1}.
\]
Агрегированная характеристика по окну длины $k$ может быть вычислена как
\[
\overline{L}_t^{(k)} = \frac{1}{k}\sum_{i=0}^{k-1} L_{t-i}.
\]

Эти признаки позволяют перевести временную структуру процесса в форму, пригодную для последующей модели.
Однако при их построении необходимо строго соблюдать направление времени и не допускать утечки будущих наблюдений.

\section{Что должен сделать обучающийся}

После выполнения главы обучающийся должен:
\begin{itemize}[leftmargin=1.25cm]
\item построить таблицу описательных статистик;
\item выделить несколько содержательных визуализаций;
\item сформировать календарные и лаговые признаки;
\item объяснить, почему выбранные признаки имеют смысл для дальнейшего анализа.
\end{itemize}
