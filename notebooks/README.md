# Ноутбуки

Этот каталог содержит:

- учебные практикумы по главам курса;
- шаблон для новых блокнотов;
- блокноты по сквозным кейсам классификации, регрессии и временных рядов.

## Текущий состав

- `01_course_setup.ipynb`
- `02_data_sources_and_quality.ipynb`

## Принцип

Каждый блокнот должен:

- читаться как академический практикум;
- вести пользователя через задачу, а не комментировать внутреннюю организацию проекта;
- быть воспроизводимым после обычного клонирования репозитория;
- быть согласованным с соответствующей главой пособия.
